Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. It is a versatile, dynamically-typed language widely used for web development, data analysis, artificial intelligence, scientific computing, and many other domains. Python emphasizes code readability with its use of significant whitespace and straightforward syntax, making it an excellent choice for both beginners and experienced developers.

Python provides an extensive standard library that includes modules for handling tasks such as file I/O, string manipulation, mathematics, and internet protocols. Additionally, Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Python's functionality can be extended with a rich ecosystem of third-party libraries and frameworks, making it suitable for a wide range of applications.

What you should already know

This guide assumes you have the following basic background:

  • A general understanding of programming concepts such as variables, loops, and functions;
  • Familiarity with text editors or IDEs used for writing code;
  • Basic knowledge of the command line for running scripts.

If you are new to programming, Python is an excellent language to start with. Consider exploring beginner-friendly tutorials or interactive environments like Jupyter Notebook to practice Python concepts.

Python and Other Languages

Python shares some characteristics with other programming languages, but it stands out for its simplicity and flexibility. Unlike statically-typed languages like Java or C++, Python uses dynamic typing, allowing you to assign different types of values to the same variable without declaring the type explicitly.

Python’s object-oriented features are optional, allowing you to use procedural or functional paradigms as needed. Its approach to object orientation uses class-based inheritance and promotes clean, modular code.

Python scripts are typically shorter and easier to write compared to their counterparts in other languages, owing to its simple syntax and powerful built-in functions.

Hello World

To write your first Python program, simply open a Python interpreter or create a .py file and write the following:

print("Hello, World!")

Run the program from the command line or an IDE to see the output. Python's print() function makes it easy to display output to the console.

Variables

In Python, variables are dynamically typed, meaning you don’t need to declare their type explicitly. Python uses meaningful names for variables and enforces proper indentation for code blocks.

x = 42 name = "Alice" is_active = True

Rules for variable names:

  • Must start with a letter or an underscore (_);
  • Cannot start with a digit;
  • Can include letters, digits, and underscores;
  • Are case-sensitive (name and Name are different).
Constants

Python doesn’t have a built-in constant declaration like const in some languages. By convention, constants are written in all uppercase letters.

PI = 3.14159 MAX_CONNECTIONS = 100

Although Python allows modification of such variables, it’s understood by convention that they should remain unchanged.

Control Structures

Python uses indentation to define code blocks, making it easy to read and maintain.

If...else statement:

if condition: # Code block if condition is True else: # Code block if condition is False x = 10 if x > 5: print("x is greater than 5") else: print("x is less than or equal to 5")
Loops

Python supports two main types of loops:

For Loop

for i in range(5): print(i) # Outputs numbers 0 through 4

While Loop

count = 0 while count < 5: print(count) count += 1
Functions

Functions in Python are defined using the def keyword, followed by a function name, parameters (optional), and a code block.

def greet(name): return f"Hello, {name}!"

You can call this function as follows:

print(greet("Alice")) # Outputs: Hello, Alice!

Python supports both positional and keyword arguments, as well as default parameter values.
